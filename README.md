# Vega AI Virtual Assistant

A smart AI-powered virtual assistant built using React and Node.js.  
It can understand voice commands, generate intelligent responses using Gemini API, and provide a smooth chat experience.

💬 Real-time chat interface  
⚡ Fast and responsive UI  
🔐 Secure API handling using .env  

---
## Voice Commands
 
| Bolo | Kya hoga |
|---|---|
| `[NAME] Instagram kholo` | Instagram open hoga |
| `[NAME] YouTube pe [song] chala do` | Song chalega |
| `[NAME] Google karo [topic]` | Google search hoga |
| `[NAME] kitna baja hai` | Time batayega |
| `[NAME] aaj kya date hai` | Date batayega |
| `[NAME] remind karo [time] pe [task]` | Reminder set hoga |
